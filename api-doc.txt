- REGISTER
POST /auth/register
BODY
  - email string required
  - password string required
  - confirm string required
  - phone string
RESPONSE
  - 201 message
  - 400 message
  - 409 message
  - 500 message

- LOGIN
POST /auth/login
BODY
  - email string required
  - password string required
RESPONSE
  - 200 access_token
  - 400 message
  - 401 message
  - 500 message

- GET AUTHENTICATED USER PROFILE
GET /auth/me
AUTHORIZATION Bearer [access_token]
RESPONSE
  - 200 user
  - 400 message
  - 401 message
  - 500 message
